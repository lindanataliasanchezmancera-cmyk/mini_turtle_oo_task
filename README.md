# Mini Turtle OO

Un paquete educativo en Python que refactoriza un sistema de dibujo tipo *turtle* usando texto (ASCII).
El proyecto aplica **Programación Orientada a Objetos**, **encapsulamiento** y elimina el uso de
variables globales.

El propósito principal es mostrar cómo varios objetos pueden mantener su propio estado de manera
independiente mediante atributos de instancia.

---

## Objetivo del Proyecto

- Transformar un diseño funcional en uno orientado a objetos
- Evitar el uso de variables globales
- Aplicar encapsulamiento usando atributos de instancia
- Permitir la creación de múltiples tortugas independientes
- Ofrecer una interfaz clara a través de una clase
